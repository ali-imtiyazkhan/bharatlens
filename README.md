# 🇮🇳 BharatLens: Immersive Cultural Tourism Platform

![BharatLens Hero](https://images.unsplash.com/photo-1548013146-72479768bada?q=80&w=2670&auto=format&fit=crop)

**BharatLens** is a next-generation heritage tourism platform designed to bridge the gap between India's ancient past and the modern traveler. Using a blend of **Generative AI**, **Real-time Social Networking**, and **Augmented Reality**, BharatLens transforms every historical monument into a living, breathing story.

---

## ✨ Core Features

### 🎙️ AI Heritage Narrator
Forget static placards. Experience high-fidelity audio guides with multi-persona support (**Wise Historian**, **Modern Guide**, or **Mystical Sage**). Narrations are dynamically generated by Gemini and synchronized with the native browser Speech API for zero-latency playback.

### 📜 Heritage Voices & Transcription
Talk to the past. Use our AI-powered chat system to converse with historical figures, or use the **Live AI Transcription** tool to record and preserve oral histories from local elders. The system automatically translates, categorizes, and archives these stories in real-time.

### 📸 AR Lens & Translation
Point your camera at any monument to identify it instantly. Use the real-time translator to decode ancient inscriptions or modern signage on the fly.

### 🏆 Gamified Exploration & Rewards
Turn your travels into a quest. Earn **HP (Health/History Points)** and **Badges** for every site you visit or story you archive. The **Real-time Engagement Loop** ensures your Token balance in the Navbar updates the second you earn points.

### 💬 Social & Community Pulse
Network with fellow explorers. Watch the **Community Pulse**—a live activity feed of shared journeys, check-ins, and archived stories from across the subcontinent. Join **Interest Communities** to plan group expeditions.

### 🗺️ AI Trip Planner
Generate optimized, budget-conscious travel itineraries from North to South in seconds, tailored to your specific interests and time constraints.

## 🎨 Premium UI & Experience

BharatLens features a professional **"Studio" Aesthetic** designed for an immersive user experience:

- **Immersive Mobile Menu**: A full-screen, portal-rendered navigation menu with high-fidelity glassmorphism and background blur.
- **Custom Dropdown System**: Replaced standard browser elements with animated, responsive `CustomDropdown` components that adapt to mobile layouts without clipping.
- **React Portal Layering**: Utilizes `react-dom/portal` to ensure critical UI overlays (menus, dropdowns) are isolated from page-specific stacking contexts, guaranteeing they always appear on the front screen.
- **Smooth Interaction Design**: Powered by `framer-motion` for fluid transitions, spring-based animations, and responsive interactive feedback.

## 🛠️ Technology Stack

BharatLens is built as a high-performance **Monorepo** using:

- **Framework**: [Next.js 14](https://nextjs.org/) (Frontend) & [Express](https://expressjs.com/) (API)
- **Monorepo Management**: [Turborepo](https://turbo.build/)
- **Web Speech API**: Zero-latency TTS & Real-time Live Transcription
- **Database**: [Prisma](https://www.prisma.io/) with [PostgreSQL (Neon)](https://neon.tech/)
- **AI Engine**: [Google Gemini Pro/Flash](https://ai.google.dev/) (Centralized via custom utility)
- **Styling**: [Framer Motion](https://www.framer.com/motion/) for premium animations
- **Localization**: Native I18n support (English/Hindi)
- **Identity**: JWT-based authentication with session synchronization

## 🌍 Production Deployment

BharatLens uses a distributed production architecture:

- **Frontend**: [Vercel](https://vercel.com/) (Next.js)
- **Backend API**: [Render](https://render.com/) (Express + WebSockets)
- **Database**: [Render PostgreSQL](https://render.com/docs/databases) (Managed DB)

### 1. Vercel Configuration (Frontend)
Set these environment variables in your Vercel project:
- `NEXT_PUBLIC_API_URL`: Use your production Render URL (e.g., `https://bharatlens-api.onrender.com`)
- `API_URL`: Same as above.

### 2. Render Configuration (Backend)
The backend is automatically deployed using the `render.yaml` blueprint. The build pipeline is optimized to:
1. Install all monorepo dependencies.
2. Synchronize the database schema with `db:push`.
3. Build the Express API.

## 🚀 Getting Started

### 1. Prerequisites
- Node.js 18+ 
- pnpm (`npm install -g pnpm`)

### 2. Installation
```bash
pnpm install
```

### 3. Environment Setup
Create a `.env` file in `apps/api/` and `apps/web/`:

```env
# Database
DATABASE_URL="postgresql://user:pass@host:6543/db?sslmode=require"

# AI
GEMINI_API_KEY="your-api-key"

# Auth
JWT_SECRET="your-secret"
```

### 4. Database Setup & Dev
```bash
# Push schema to DB
pnpm --filter @bharatlens/db run db:push

# Generate client
pnpm --filter @bharatlens/db run db:generate
```

### 5. Run Development Server
```bash
pnpm dev
```
Visit [http://localhost:3000](http://localhost:3000) to start your journey.
