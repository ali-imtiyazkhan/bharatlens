# 🇮🇳 BharatLens: Immersive Cultural Tourism Platform

![BharatLens Hero](https://images.unsplash.com/photo-1548013146-72479768bada?q=80&w=2670&auto=format&fit=crop)

**BharatLens** is a next-generation heritage tourism platform designed to bridge the gap between India's ancient past and the modern traveler. Using a blend of **Generative AI**, **Real-time Social Networking**, and **Augmented Reality**, BharatLens transforms every historical monument into a living, breathing story.

---

## ✨ Core Features

### 🎙️ AI Heritage Narrator
Forget static placards. Listen to "Insider Stories" dynamically generated by AI historians. Every monument features a premium audio guide that uncovers architectural secrets and untold legends.

### 📜 Heritage Voices
Talk to the past. Use our AI-powered chat system to converse with historical figures like **Akbar the Great** or **Rani Laxmibai**. Now with **Voice Synthesis**, history literally speaks to you.

### 📸 AR Lens & Translation
Point your camera at any monument to identify it instantly. Use the real-time translator to decode ancient inscriptions or modern signage on the fly.

### 🏆 Gamified Exploration
Turn your travels into a quest. Earn **XP, Badges, and Tokens** for every site you visit. Compete on the **Elite Historians Leaderboard** and unlock exclusive digital collectibles.

### 💬 Social & Communities
Network with fellow explorers. Start **Real-time Chats** via Socket.io or join **Interest Communities** (e.g., "Mughal Architecture Buffs") to plan group expeditions.

### 🗺️ AI Trip Planner
Generate optimized, budget-conscious travel itineraries from North to South in seconds, tailored to your specific interests and time constraints.

## 🛠️ Technology Stack

BharatLens is built as a high-performance **Monorepo** using:

- **Framework**: [Next.js 14](https://nextjs.org/) (Frontend) & [Express](https://expressjs.com/) (API)
- **Monorepo Management**: [Turborepo](https://turbo.build/)
- **Real-time**: [Socket.io](https://socket.io/) for instant messaging
- **Database**: [Prisma](https://www.prisma.io/) with [PostgreSQL (Neon)](https://neon.tech/)
- **AI Engine**: [Google Gemini Pro/Flash](https://ai.google.dev/) for narrations and chat
- **Styling**: [Framer Motion](https://www.framer.com/motion/) for premium animations
- **Identity**: JWT-based authentication with local persistence

## 🚀 Getting Started

### 1. Prerequisites
- Node.js 18+ 
- pnpm (`npm install -g pnpm`)

### 2. Installation
```bash
pnpm install
```

### 3. Environment Setup
Create a `.env` file in `apps/api/` and `apps/web/`:

```env
# Database
DATABASE_URL="postgresql://user:pass@host:6543/db?sslmode=require&pgbouncer=true"

# AI
GEMINI_API_KEY="your-api-key"

# Auth
JWT_SECRET="your-secret"
```

### 4. Database Initialization
```bash
pnpm --filter @bharatlens/db run db:push
pnpm --filter @bharatlens/db run db:generate
```

### 5. Run Development Server
```bash
pnpm dev
```
Visit [http://localhost:3000](http://localhost:3000) to start your journey.
